# Cafe Delight - Café Menu

## Overview

Cafe Delight is a simple frontend project created using HTML and CSS. It displays a café menu featuring a variety of beverages, desserts, and snacks along with their prices. The project focuses on basic web design concepts such as page structure, styling, and layout creation.

## Features

* Café menu interface
* Organized food and beverage sections
* Clean and simple design
* Price listing for menu items
* Built using only HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
├── index.html
├── style.css
```

## Purpose

This project was developed as a beginner-level frontend practice project to improve HTML and CSS skills and understand how web pages are structured and styled.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
